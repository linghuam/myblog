# 浏览器回流与重绘

页面渲染流程：

1. HTML --> DOM 树
2. CSS --> CSSOM 树
3. DOM 树 + CSSOM 树 --> render 树
4. 绘制页面

回流（reflow）：
当`render tree`中的一部分（或全部）因为元素的**规模尺寸，布局，隐藏**等改变而需要**重新构建**。

重绘（redraw）：
当`render tree`中的一些元素需要更新属性，而这些属性只是**影响元素的外观、风格，而不会影响布局的**，比如`background-color`。


**回流必将引起重绘，而重绘不一定会引起回流。**
